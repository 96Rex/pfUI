# pfUI

适用于魔兽世界(乌龟服)的一个插件，旨在完全取代原来的用户界面。

## 如何安装 (手动)
1. 下载 **[最新版本](https://github.com/96rex/pfUI/archive/master.zip)**
2. 解压压缩包
3. 将"pfUI-master"文件夹重命名为"pfUI"
4. 复制"pfUI"文件夹到 游戏目录\Interface\AddOns
5. 重启游戏

## 命令行

    /pfui         打开设置界面
    /share        打开配置分享界面
    /farm         切换为刷怪模式
    /pfcast       鼠标悬停施法
    /focus        为当前目标单位创建焦点单位框架
    /castfocus    焦点单位施法
    /clearfocus   清除焦点单位框架
    /swapfocus    切换焦点目标单位
    /pftest       切换为调试模式
    /abp          插件按钮面板

## 与shagu原版不同的地方

* 移除多余语言
* 移除多客户端支持,仅支持乌龟服
* 更美观的默认配置(即将实现)
* 更多为乌龟服专属的界面皮肤(即将实现)
