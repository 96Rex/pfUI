# pfUI

适用于魔兽世界1.12.1和2.4.3版本的一个插件，旨在完全取代原来的用户界面。这个设计的灵感来自于我在TukUI， ElvUI和其他网站上看到的几个截图。这个插件提供了现代的功能和简约的风格，极易上手。它完全从头开始编写，没有包含任何第三方插件或函数库。

## 如何安装 (手动)
1. 下载 **[最新版本](https://github.com/96rex/pfUI/archive/master.zip)**
2. 解压压缩包
3. 将"pfUI-master"文件夹重命名为"pfUI"
4. 复制"pfUI"文件夹到 游戏目录\Interface\AddOns
5. 重启游戏

## 命令行

    /pfui         打开设置界面
    /share        打开配置分享界面
    /gm           打开GM表单界面
    /rl           重载用户界面
    /farm         切换为刷怪模式
    /pfcast       鼠标悬停施法
    /focus        为当前目标单位创建焦点单位框架
    /castfocus    焦点单位施法
    /clearfocus   清除焦点单位框架
    /swapfocus    切换焦点目标单位
    /pftest       切换为调试模式
    /abp          插件按钮面板
